Java基础

ClassLoader（类加载机制）

Java类都需要经过JVM加载才能运行，ClassLoader的主要作用就是Java类文件的加载

ClassLoader类有如下核心方法：

  1. loadClass（加载指定的Java类）
  2. findClass（查找指定的Java类）
  3. findLoadedClass（查找JVM已经加载过的类）
  4. defineClass（定义一个Java类）
  5. resolveClass（链接指定的Java类）

URLClassLoader

URLClassLoader继承了ClassLoader，且提供了加载远程资源的能力，在写漏洞利用的payload或者webshell的时候可以加载远程的jar来实现远程的类方法调用
